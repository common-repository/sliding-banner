== Description ==
 
Sliding Banner allows you to show differents banners in the bottom of the page with Offers, News, or custom content for promotion.

When you Install the plugin you can create your owns banners in the "Banners" section. After that, you have to activate the Sliding Banner in the plugin options.

Plugin Options are in Settings -> Sliding Banner. Custom options are:

- Activate Sliding Banner
- Activate just on responsive version
- Show Sliding Banner just on Homepage
- Animation duration.

== Installation ==
 
1. Upload `sliding-banner-news-and-offers` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the wordpress admin menu create your Banners in "Banners" section.
4. Configure de plugin in the Settings -> Sliding Banner section.
